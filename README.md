# Contact_book

聯絡人管理系統 - 一個美觀且易用的 Qt 通訊錄應用程式

## 功能特色

### 📋 基本功能
- **新增聯絡人**：輸入學號、班級、姓名和電話資訊
- **刪除聯絡人**：選擇並刪除不需要的聯絡人
- **匯入資料**：從文字檔案匯入聯絡人清單
- **匯出資料**：將聯絡人資料儲存到文字檔案
- **排序功能**：點擊表格標題可排序資料

### 🎨 介面特色
- **現代化設計**：採用 Material Design 風格的配色方案
- **清晰的視覺層次**：使用不同顏色區分按鈕功能
  - 綠色：新增聯絡人
  - 紫色：刪除聯絡人
  - 橘色：匯入資料
  - 藍色：匯出資料
  - 紅色：結束程式
- **圖示按鈕**：使用 emoji 圖示讓功能更直觀
- **工具提示**：滑鼠停留在按鈕上會顯示功能說明
- **輸入提示**：每個輸入框都有提示文字
- **交替行色**：表格採用交替背景色，更易閱讀
- **圓角設計**：按鈕和輸入框採用圓角設計，更加美觀
- **懸停效果**：按鈕有懸停和按下的視覺回饋

### 🔧 使用者體驗改善
- **輸入驗證**：防止新增空白的聯絡人資料
- **自動聚焦**：新增聯絡人後，游標自動回到第一個輸入欄位
- **整列選取**：點擊表格任何位置即可選取整列
- **適當的視窗大小**：最小視窗大小為 900x650，確保所有內容可見
- **適當的欄位寬度**：各欄位設定適當寬度，避免資訊被截斷

## 編譯與執行

### 需求
- Qt 5.x 或更高版本
- C++17 支援
- qmake 或 Qt Creator

### 編譯步驟
```bash
qmake Contact_book.pro
make
```

### 使用 Qt Creator
1. 開啟 `Contact_book.pro`
2. 設定編譯套件
3. 建置並執行

## 檔案格式

聯絡人資料以 CSV 格式儲存，每行包含四個欄位：
```
學號,班級,姓名,電話
```

範例：
```
A12345,資訊一甲,王小明,0912345678
B23456,資訊二乙,李小華,0923456789
```

## 操作說明

1. **新增聯絡人**
   - 在下方輸入欄位填寫資訊
   - 點擊「➕ 新增聯絡人」按鈕
   - 資料會立即顯示在表格中

2. **刪除聯絡人**
   - 在表格中點選要刪除的聯絡人（可多選）
   - 點擊「🗑️ 刪除」按鈕

3. **匯入資料**
   - 點擊「📁 匯入」按鈕
   - 選擇要匯入的文字檔案
   - 資料會自動載入到表格

4. **匯出資料**
   - 點擊「💾 匯出」按鈕
   - 選擇儲存位置和檔名
   - 表格資料會儲存成文字檔

5. **排序資料**
   - 點擊表格標題列（學號、班級、姓名、電話）
   - 資料會依該欄位排序

6. **結束程式**
   - 點擊「❌ 結束」按鈕
   - 程式會自動儲存後關閉
